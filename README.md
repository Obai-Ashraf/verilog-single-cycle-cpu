# Single Cycle Processor (Verilog)

## 📌 Description
This project is a Verilog implementation of a Single-Cycle Processor developed as part of the Computer Architecture course. The processor executes instructions in a single clock cycle and demonstrates basic CPU design concepts.

---

## 🧠 Project Overview
The design follows a simplified CPU architecture and includes key components such as:
- Program Counter (PC)
- Instruction execution flow (Single-cycle design concept)
- Basic datapath structure
